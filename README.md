# To-do App

Простой pet-project для изучения API-автотестов и дальнейших UI-автотестов.

## Стек
- Flask
- Flask-SQLAlchemy
- PostgreSQL
- Flask-JWT-Extended
- HTML / CSS / JS

## Установка

1. Создать и активировать виртуальное окружение
2. Установить зависимости:
   pip install -r requirements.txt

3. Скопировать `.env.example` в `.env`
4. Создать базу данных PostgreSQL `todo_db`
5. Запустить проект:
   python run.py

## Что уже реализовано
- регистрация
- логин
- сохранение JWT на клиенте
- базовый UI
